Comando para CSS EXTERNO na tag Head:
<link rel="stylesheet" href="nomeestilo.css">




Comando INLINE para cor de fundo:
style="background-color: nomecor;"



Comando INTERNO para cor de fundo:
<style>
	body{
		background-color: nomecor;
	}
</style>



Comando EXTERNO para cor de fundo:
body{
	background-color: nomecor;
}




Comando INLINE para cor de fonte:
style="color: nomecor;"



Comando INTERNO para cor de fonte:
<style>
	h1{
		color: nomecor;
	}
</style>



Comando EXTERNO para cor da fonte:
h1{
	color: nomecor;	
}




Comando INLINE para tipo da fonte:
style="font-family: nomefonte;"



Comando INTERNO para tipo da fonte:
<style>
	body{
		font-family: nomefonte;
	}
</style



Comando EXTERNO para tipo da fonte:
body{
	font-family: nomefonte;
}
